## 3. 输入接口定义 (VAR_INPUT) - 52个

### 3.1 系统控制信号组 (3个BOOL)

#### 3.1.1 i_b使能

| 属性 | 值 |
|------|-----|
| **针脚名称** | `i_b使能` |
| **数据类型** | BOOL |
| **功能描述** | 总使能信号，控制本功能块的运行/停止 |
| **有效值域** | TRUE=允许运行, FALSE=停止并清零全部输出 |
| **默认行为** | FALSE时强制归零所有输出（6气缸+3伺服+下游信号），状态机和计数器复位 |
| **来源** | 主控系统就绪标志（急停解除、安全门关闭等条件满足后置TRUE） |
| **更新周期** | 每扫描周期 |
| **优先级** | 最高（覆盖所有其他输入） |

#### 3.1.2 i_b自动模式

| 属性 | 值 |
|------|-----|
| **针脚名称** | `i_b自动模式` |
| **数据类型** | BOOL |
| **功能描述** | 自动运行模式选择信号 |
| **有效值域** | TRUE=选中自动模式, FALSE=未选中 |
| **互斥关系** | 与`i_b手动模式`互斥（不能同时为TRUE，由HMI保证） |
| **来源** | HMI M区变量 |

**重要约束**: 若`i_b自动模式`与`i_b手动模式`同时为FALSE，则功能块进入安全停机状态。

#### 3.1.3 i_b手动模式

| 属性 | 值 |
|------|-----|
| **针脚名称** | `i_b手动模式` |
| **数据类型** | BOOL |
| **功能描述** | 手动调试模式选择信号 |
| **有效值域** | TRUE=选手动模式, FALSE=未选中 |
| **互斥关系** | 与`i_b自动模式`互斥 |
| **来源** | HMI M区变量 |

### 3.2 操作按钮组 (3个BOOL)

#### 3.2.1 i_b启动

| 属性 | 值 |
|------|-----|
| **针脚名称** | `i_b启动` |
| **数据类型** | BOOL |
| **功能描述** | 自动循环启动按钮信号 |
| **触发方式** | **上升沿触发**（建议主控做R_TRIG边沿检测） |
| **生效条件** | 仅在`i_b自动模式=TRUE`且`i_b使能=TRUE`且无故障时有效 |
| **典型用法** | 按下后状态机从步骤0(空闲)进入步骤1(等待放料完成) |

#### 3.2.2 i_b停止

| 属性 | 值 |
|------|-----|
| **针脚名称** | `i_b停止` |
| **数据类型** | BOOL |
| **功能描述** | 自动循环停止按钮信号 |
| **触发方式** | **电平有效**（按下即停止） |
| **行为描述** | 按下后状态机立即回到步骤0(空闲)，清除所有运动请求，保持当前层数进度 |

#### 3.2.3 i_b复位

| 属性 | 值 |
|------|-----|
| **针脚名称** | `i_b复位` |
| **数据类型** | BOOL |
| **功能描述** | 故障复位/初始化按钮信号 |
| **触发方式** | **电平有效**（按下执行复位） |
| **行为描述** | 清除报警锁存、归零所有输出、重新执行初始化流程（伺服使能检查） |

```
IF s_i已完成层数 < 2 THEN

    b条件 := i_输送机_L3_放料完成 AND i_输送机_L4_放料完成;

```

### A.2 报警码速查

```
报警代码速查 (101~108段):

  0   = 无报警 (正常)
  101 = Z轴伺服故障 (最高优先级!)
  102 = X1轴伺服故障 (最高优先级!)
  103 = 升降下降超时 (步骤2/7)
  104 = 升降上升超时 (步骤5/9)
  105 = 夹紧超时     (步骤3)
  106 = 松开超时     (步骤8)
  107 = 产品检测失败 (步骤4, 4传感器AND判断)
  108 = X1轴移动超时 (步骤6)

处理方法:
  101/102 → 排除伺服故障 → 按复位
  103~106,108 → 排除原因 → 按复位
  107      → 检查边框/传感器 → 按复位
```
