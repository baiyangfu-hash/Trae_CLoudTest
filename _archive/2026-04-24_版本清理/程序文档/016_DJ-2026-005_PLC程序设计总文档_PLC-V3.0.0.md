## 2. 项目概述

### 2.1 系统功能
边框缓存机是一种自动化设备，用于存储和管理边框材料。主要功能包括：
- 多层存储：4层存储结构，每层可独立运行
- 自动分料：通过分料气缸和输送带实现自动分料
- 自动取放料：通过取料机构和伺服系统实现自动取放料
- 安全保护：急停、安全门等安全功能
- 与打胶机交互：实现与打胶机的自动对接

### 2.2 系统架构
系统采用三层架构设计：
- **设备层**：直接控制硬件设备，如伺服、气缸、输送带等
- **逻辑层**：实现分料、取放料等核心逻辑
- **主控层**：协调各层工作，实现系统整体控制

#### 4.3.1 分料逻辑状态机
| 状态 | 描述 | 转换条件 |
|------|------|----------|
| FEED_IDLE | 空闲状态 | 启动信号触发，进入FEED_BLOCK_DOWN |
| FEED_BLOCK_DOWN | 阻挡下降 | 阻挡气缸到位，进入FEED_CONVEY_FORWARD |
| FEED_CONVEY_FORWARD | 输送前进 | 输送时间到，进入FEED_STOP_CONVEY |
| FEED_STOP_CONVEY | 停止输送 | 输送停止，进入FEED_BLOCK_UP |
| FEED_BLOCK_UP | 阻挡上升 | 阻挡气缸到位，进入FEED_FEED_OUT |
| FEED_FEED_OUT | 分料推出 | 分料气缸到位，进入FEED_RESET |
| FEED_RESET | 复位状态 | 分料气缸复位，进入FEED_IDLE |

| PICK_CLAMP | 取料夹紧 | 夹紧到位，进入PICK_LIFT_UP |
| PICK_LIFT_UP | 取料上升 | 上升到位，进入PICK_MOVE |
| PICK_MOVE | 移动到放料位置 | 移动到位，进入PLACE_LIFT_DOWN |
| PLACE_LIFT_DOWN | 放料下降 | 下降到位，进入PLACE_RELEASE |
| PLACE_RELEASE | 放料松开 | 松开到位，进入PLACE_LIFT_UP |

#### 4.3.3 系统控制状态机

|------|------|----------|

## 5. 程序功能详解

### 5.1 主程序流程
1. **初始化**：调用FB_Main_Initialization进行系统初始化
2. **IO映射**：调用FB_Device_IOMapping实现IO映射
3. **安全管理**：调用FB_Main_SafetyManager进行安全监控
4. **系统控制**：调用FB_Main_SystemControl管理系统状态
5. **层协调**：调用FB_Main_LayerCoordinator协调各层工作
6. **分料逻辑**：调用FB_Logic_LayerFeed实现分料功能
7. **取放料逻辑**：调用FB_Logic_PickPlace实现取放料功能
8. **输送控制**：调用FB_Logic_ConveyorControl控制输送线
9. **伺服控制**：调用FB_Logic_ServoControl控制伺服系统
10. **打胶机交互**：调用FB_Logic_GlueMachineInterface实现与打胶机的交互
11. **报警管理**：调用FB_Main_AlarmManager处理报警信息

### 5.2 分料功能
- **功能**：将边框材料从输送线分离到指定位置
- **实现**：通过FB_Logic_LayerFeed功能块，使用状态机控制阻挡气缸、分料气缸和输送线的动作
- **流程**：
  1. 检测到材料到达分料位置
  2. 阻挡气缸下降，停止材料
  3. 输送线启动，将材料输送到分料位置
  4. 阻挡气缸上升
  5. 分料气缸推出，将材料分离
  6. 分料气缸复位，准备下一次分料

### 5.3 取放料功能
- **功能**：将边框材料从分料位置抓取并放置到指定位置
- **实现**：通过FB_Logic_PickPlace功能块，使用状态机控制取料机构的升降、抓取和移动动作
- **流程**：
  1. 取料机构移动到取料位置
  2. 取料机构下降
  3. 夹紧气缸夹紧材料
  4. 取料机构上升
  5. 取料机构移动到放料位置
  6. 取料机构下降
  7. 夹紧气缸松开，放置材料
  8. 取料机构上升，准备下一次取放料

### 5.4 输送控制功能
- **功能**：控制输送线的正转、反转、慢速等动作
- **实现**：通过FB_Logic_ConveyorControl功能块，控制变频器的输入信号
- **流程**：
  1. 接收启动信号
  2. 根据控制要求，输出正转、反转或慢速信号
  3. 监控输送线状态
  4. 接收停止信号，停止输送线

### 5.5 伺服控制功能
- **功能**：控制伺服轴的回原点、定位、JOG等动作
- **实现**：通过FB_Logic_ServoControl功能块，控制伺服驱动器的使能、脉冲和方向信号
- **流程**：
  1. 回原点：控制伺服轴回到原点位置
  2. 定位：控制伺服轴移动到指定位置
  3. JOG：控制伺服轴以指定速度运行
  4. 监控伺服轴状态，处理故障情况

### 5.6 安全功能
- **功能**：监控急停、安全门等安全信号，确保系统安全运行
- **实现**：通过FB_Main_SafetyManager功能块，监控安全信号并处理安全故障
- **流程**：
  1. 监控急停按钮状态
  2. 监控安全门状态
  3. 当检测到安全故障时，停止系统运行
  4. 当安全故障消除后，允许系统重新启动

### 5.7 报警功能
- **功能**：检测系统故障并触发报警
- **实现**：通过FB_Main_AlarmManager功能块，检测故障信号并处理报警信息
- **流程**：
  1. 监控系统故障信号
  2. 当检测到故障时，触发报警
  3. 记录报警信息
  4. 当故障消除后，清除报警

### 5.8 与打胶机交互功能
- **功能**：实现与打胶机的自动对接
- **实现**：通过FB_Logic_GlueMachineInterface功能块，与打胶机进行信号交互
- **流程**：
  1. 接收打胶机的允许送料信号
  2. 当材料准备就绪时，向打胶机发送允许抓料信号
  3. 接收打胶机的取料完成信号
  4. 继续下一循环的分料和取放料操作

### 8.1 常见故障及处理

| 故障现象 | 可能原因 | 处理方法 |
|----------|----------|----------|
| 系统无法启动 | 急停按钮未复位 | 旋转急停按钮，按下复位按钮 |
| 安全门未关闭 | 关闭安全门，按下复位按钮 |
| 电源故障 | 检查电源连接 |
| 分料气缸不动作 | 气缸故障 | 检查气缸是否损坏，更换气缸 |
| 电磁阀故障 | 检查电磁阀是否损坏，更换电磁阀 |
| 气源压力不足 | 检查气源压力，确保压力正常 |
| 取料机构不动作 | 伺服故障 | 检查伺服驱动器报警信息，排除故障 |
| 气缸故障 | 检查气缸是否损坏，更换气缸 |
| 输送线不运行 | 变频器故障 | 检查变频器报警信息，排除故障 |
| 电机故障 | 检查电机是否损坏，更换电机 |
| 与打胶机通信故障 | 通信线缆故障 | 检查通信线缆连接，更换线缆 |
| 打胶机故障 | 联系打胶机维护人员，排除故障 |

### 8.2 维护建议
1. **定期检查**：
   - 检查所有气缸的工作状态，确保气缸动作顺畅
   - 检查所有传感器的工作状态，确保传感器信号正常
   - 检查所有伺服驱动器的工作状态，确保伺服系统运行正常
   - 检查所有变频器的工作状态，确保输送线运行正常
   - 检查所有安全设备的工作状态，确保安全系统运行正常

2. **定期润滑**：
   - 定期润滑气缸的运动部件
   - 定期润滑伺服轴的运动部件
   - 定期润滑输送线的运动部件

3. **定期清洁**：
   - 定期清洁传感器的感应面
   - 定期清洁气缸的活塞杆
   - 定期清洁输送线的表面

4. **故障记录**：
   - 记录故障发生的时间、现象和处理方法
   - 分析故障原因，采取预防措施
