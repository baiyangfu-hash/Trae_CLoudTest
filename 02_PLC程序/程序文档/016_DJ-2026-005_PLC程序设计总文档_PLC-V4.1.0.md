## 3. 项目概述

### 3.1 系统功能

边框缓存机是一种自动化设备，用于存储和管理边框材料。主要功能包括：

| 功能类别 | 详细描述 |
|----------|----------|
| **多层存储** | 4层独立存储结构，每层可独立运行分料/输送循环 |
| **自动分料** | 通过阻挡气缸 + 分料气缸 + 输送带的配合实现精确分料 |
| **自动取放料** | Z轴升降 + X1轴横移 + 前后夹爪协同工作，一次取两根边框 |
| **4层循环取料** | 第1次取L1+L2层，第2次取L3+L4层，共2次循环完成全部取料 |
| **安全保护** | 急停按钮、双安全门锁定、安全继电器回路 |
| **与打胶机交互** | 通过硬接线信号(X75-X79,X102/Y50,Y51,Y44,Y47)实现自动对接，含故障/急停/通讯异常联锁 |
| **与组框机交互** | 预留接口(7DI+5DO)，含安全联锁/开门互锁/送料请求协调，地址待现场确定 |
| **外部设备报警** | 300段报警码(打胶机300~302, 组框机310~312)，纳入公共报警FB统一管理 |
| **HMI人机界面** | 模式选择(自动/手动)、启停控制、参数设定、状态监控、报警显示 |
| **MES数据接口** | D406~D425提供最近10条去重报警队列供MES系统读取上传 |

### 3.2 核心工艺流程

```
                    完整工艺流程 (V4.1.0架构视角 - 含外部设备交互+规范修订)
                    ======================================================

  [组框机(上游)] ←→ [边框缓存机(本设备)] ←→ [打胶机(下游)]
       ↑ 预留接口          ↓                    ↑ 已定义接口
   (7DI+5DO)        [四层输送机] → [取放料机构] → [打胶机送料机构]
                    (分料/输送)    (Z升降+X1横移)    (X2横移+交互)

  外部设备交互协议:
  ┌───────────┐  控制信号  ┌──────────────┐  控制信号  ┌───────────┐
  │  组框机   │ ◄──────► │  边框缓存机   │ ◄──────► │  打胶机   │
  │ (上游)    │          │   (本设备)    │          │ (下游)    │
  └───────────┘          └──────────────┘          └───────────┘
     输入:              主控Step3处理:
     ·自动中             ·打胶机有效性检查+安全回退
     ·允许送料           ·组框机急停联锁(停止本机)
     ·有料请求           ·开门互锁(暂停+确认)
     ·开门请求           ·就绪信号生成
     ·急停/安全异常      ·送料请求协调(激活)
     ·通讯异常           ·外部设备报警汇总(300段)
     输出:
     ·请求送料
     ·暂停/急停回传
     ·申请开门
     ·就绪

  四层输送机 (每层独立9步):
    材料到达 → 阻挡下降 → 输送正转 → 到位检测 → 阻挡上升 → 分料推出 → 慢速送出 → 放料完成

  取放料机构 (全局11步, 一次取两层):
    等待放料完成(L1+L2或L3+L4) → Z轴下降 → 夹爪夹紧(前后同时) → 产品检测(4传感器)
    → Z轴上升 → X1轴后移到放料位 → Z轴下降 → 夹爪松开 → Z轴上升 → 放料完成通知

  打胶机送料机构 (6步):
    等待取放料放料完成 → X2轴前移到取料位 → 等待打胶机允许送料(X76)
    → 发送允许抓料(Y44) → 等待打胶机取料完成(X102) → X2轴返回待机位 → 循环
```

### 5.2 主控程序执行流程 (每个扫描周期严格按11步顺序)

```
  ┌─────────────────────────────────────────────────────────────┐
  │                  主控程序扫描周期执行顺序 (V4.1.0)             │
  │                                                             │
  │  Step1: IO映射输入  (X地址 -> 逻辑变量)                       │
  │    ├─ CPU本体: X0-X17 -> 伺服/安全/急停/复位变量             │
  │    ├─ 打胶机信号: X75,X76,X102 -> 交互变量(含扩展)           │
  │    ├─ 组框机信号(预留): 全部FALSE, 待现场激活                │
  │    ├─ DI3模块: X0-X17 -> L3/L4层28个传感器数组               │
  │    ├─ DI4模块: X0-X17 -> L1/L2层28个传感器数组               │
  │    └─ Remote_IO: X0-XF -> 取放料14个传感器                   │
  │                         ↓                                   │
  │  Step2: HMI数据读取 (M/D区 -> 逻辑变量)                      │
  │    ├─ M0-M4: 模式选择/启停/复位                              │
  │    ├─ M10-M49: 40个手动操作信号                             │
  │    └─ D100-D112: 7个工艺参数设定                             │
  │                         ↓                                   │
  │  Step3: 外部设备信号处理 (打胶机+组框机交互)                  │
  │    ├─ 3.1 打胶机有效性检查 + 通讯异常安全回退                 │
  │    ├─ 3.2 组框机安全联锁(急停→停止本机) + 开门互锁           │
  │    ├─ 3.3 本机就绪信号生成(自动+使能+无报警+非紧急停止)       │
  │    ├─ 3.4 送料请求协调逻辑(已激活)                           │
  │    ├─ 3.5 打胶机请求运行/复位请求触发逻辑                    │
  │    └─ 3.6 外部设备报警码汇总(300~312段)                     │
  │                         ↓                                   │
  │  Step3.5: 系统使能计算                                        │
  │    └─ 安全条件: 急停+双门锁+打胶机正常+组框机正常            │
  │                         ↓                                   │
  │  Step4: 调用 fb四层输送机 (i_b使能 = 系统使能 AND NOT组框机急停)│
  │  Step5: 调用 fb取放料机构 (同上联锁)                          │
  │  Step6: 调用 fb打胶机送料机构 (同上联锁)                      │
  │  Step7: 调用 fb公共报警 (含外部设备300段报警)                  │
  │  Step8: IO映射输出 (逻辑变量 -> Y地址)                        │
  │    ├─ CPU本体: Y0/Y4/Y10/Y13 <- 伺服/安全                   │
  │    ├─ 打胶机输出: Y50(请求运行),Y51(复位),Y44,Y47          │
  │    ├─ DO1: Y20-Y29 <- L3/L4层16个执行器                     │
  │    ├─ DO2: Y32-Y41 <- L1/L2层16个执行器                     │
  │    └─ Remote_IO: Y200-Y205 <- 取放料6个气缸                 │
  │                         ↓                                   │
  │  Step9: HMI状态回写 (M12/M13避开手动操作区)                   │
  │  Step10: 报警D区回写 (公共报警输出 -> D400起始)              │
  │                                                             │
  └─────────────────────────────────────────────────────────────┘
```

```
  安全系统逻辑 (在主控程序Step3.5中实现, V4.1.0增强版)
  =====================================================

  ┌─────────────────────────────────────────────────────┐
  │  本机安全条件:                                       │

  │                                                     │
  │  外部设备安全联锁:                                    │
  │  4. 打胶机无急停 (NOT i_b打胶机_急停)                │
  │  5. 打胶机无故障 (NOT i_b打胶机_故障)                 │
  │  6. 组框机无急停 (NOT i_b组框机_急停) [预留]          │
  │                                                     │
  │  → 条件满足: b系统使能 = TRUE                        │
  │    b安全继电器启动 = TRUE (Y13 ON)                   │
  │    bHMI_系统就绪 = TRUE                              │
  │                                                     │
  │  → 条件不满足: b系统使能 = FALSE                     │
  │    所有工站因i_b使能=FALSE而停止                      │
  └─────────────────────────────────────────────────────┘
```
