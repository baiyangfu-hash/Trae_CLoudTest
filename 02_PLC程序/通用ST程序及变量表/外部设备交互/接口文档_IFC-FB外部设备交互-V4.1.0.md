## 2. 功能块概述

### 2.1 功能描述

外部设备交互功能块负责处理与外部设备的所有交互逻辑，包括：
- 组框机交互：接收组框机信号，发送控制信号
- 打胶机交互：接收打胶机信号，发送控制信号
- 机器人交互：接收机器人信号，发送控制信号（预留）
- 安全联锁：处理外部设备的安全联锁逻辑
- 报警管理：汇总外部设备报警信息

### 2.2 架构特点

- **纯逻辑块**：无物理地址依赖，所有输入/输出均为逻辑变量
- **高内聚**：集中处理所有外部设备交互逻辑
- **低耦合**：与主控程序通过明确的接口交互
- **可扩展**：预留机器人接口，支持未来功能扩展

## 4. 接口使用说明

### 4.1 调用示例

```pascal
fb外部设备交互(

    i_b启动               := bHMI_启动,
    i_b停止               := bHMI_停止,
    i_b复位               := bHMI_复位 OR b复位按钮,
    
    (* 组框机输入信号 *)
    i_b组框机_自动中      := i_b组框机_自动中,
    i_b组框机_允许送料    := i_b组框机_允许送料,
    i_b组框机_有料请求    := i_b组框机_有料请求,
    i_b组框机_开门请求    := i_b组框机_开门请求,
    i_b组框机_急停        := i_b组框机_急停,
    i_b组框机_安全异常    := i_b组框机_安全异常,
    i_b组框机_通讯异常    := i_b组框机_通讯异常,
    
    (* 打胶机输入信号 *)
    i_b打胶机_自动中      := i_b打胶机_自动中,
    i_b打胶机_允许送料    := i_b打胶机_允许送料,
    i_b打胶机_取料完成    := i_b打胶机_取料完成,
    i_b打胶机_故障        := i_b打胶机_故障,
    i_b打胶机_急停        := i_b打胶机_急停,
    i_b打胶机_通讯异常    := i_b打胶机_通讯异常,
    
    (* 机器人输入信号 (预留) *)
    i_b机器人_自动中      := FALSE,   // 占位符: 待分配输入
    i_b机器人_码料完成    := FALSE,   // 占位符: 待分配输入
    i_b机器人_故障        := FALSE,   // 占位符: 待分配输入
    i_b机器人_急停        := FALSE,   // 占位符: 待分配输入
    i_b机器人_通讯异常    := FALSE,   // 占位符: 待分配输入
    
    (* 本机状态信号 *)

    i_b任何报警激活      := bHMI_任何报警激活,
    i_b系统故障          := bHMI_系统故障,
    
    (* 组框机输出信号 *)

    q_b组框机_暂停          => q_b组框机_暂停,
    q_b组框机_急停          => q_b组框机_急停,
    q_b申请开门             => q_b申请开门,
    q_b组框机_就绪          => q_b组框机_就绪,

    q_b机器人_停止码料    => ,  // 占位符: 待分配输出

    q_i外部设备报警汇总   => i_外部设备报警汇总,
    q_b系统安全条件满足    =>   // 可用于安全系统

```

### 4.2 信号处理逻辑

1. **打胶机信号有效性检查**：当通讯异常时，强制将关键输入信号置为保守值
2. **组框机安全联锁**：组框机急停/安全异常时，本机必须立即停止
3. **送料请求协调**：当本机需要边框且就绪且上游允许时，请求送料
4. **打胶机控制**：根据系统状态生成打胶机控制信号
5. **机器人控制**：预留机器人控制逻辑
6. **报警管理**：汇总外部设备报警信息，生成报警码
7. **安全条件计算**：检查所有外部设备的安全状态

### 6.2 注意事项

- 所有输入信号必须通过主控程序的IO映射传入，功能块内部不直接使用物理IO地址
- 预留的机器人接口在未来版本中会启用
- 外部设备通讯异常时，功能块会自动将信号置为安全状态

## 7. 故障处理

### 7.1 常见故障及处理

| 故障现象 | 可能原因 | 处理方法 |
|----------|----------|----------|
| 打胶机无响应 | 通讯异常 | 检查通讯线路和设备状态 |
| 组框机无法送料 | 安全条件不满足 | 检查组框机急停和安全门状态 |
| 外部设备报警 | 设备故障 | 检查对应设备的故障状态 |

### 7.2 故障恢复

- 外部设备故障排除后，通过复位按钮恢复系统
- 通讯异常恢复后，系统会自动恢复正常操作
