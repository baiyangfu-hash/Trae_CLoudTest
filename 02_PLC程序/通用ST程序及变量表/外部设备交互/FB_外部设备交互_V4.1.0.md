# 外部设备交互功能块文档

## 1. 功能块信息

| 项目 | 内容 |
|------|------|
| 功能块名称 | FB_外部设备交互 |
| 版本号 | V4.1.0 |
| 创建日期 | 2026-04-27 |
| 功能描述 | 处理与组框机、打胶机和机器人的交互逻辑 |
| 遵循规范 | 801_PLC变量命名与功能块命名规范_DEV-V1.0.5 |

## 2. 功能概述

外部设备交互功能块负责处理与外部设备的所有交互逻辑，包括：

- **组框机交互**：接收组框机信号，发送控制信号
- **打胶机交互**：接收打胶机信号，发送控制信号
- **机器人交互**：接收机器人信号，发送控制信号（预留）
- **安全联锁**：处理外部设备的安全联锁逻辑
- **报警管理**：汇总外部设备报警信息

## 4. 功能说明

### 4.1 打胶机信号处理

- **通讯异常处理**：当打胶机通讯异常时，强制将关键输入信号置为保守值（禁止状态），并关闭安全区
- **请求运行**：当本机自动启动且打胶机未在自动运行中时，请求打胶机启动运行
- **复位请求**：当本机复位且打胶机有故障或急停时，向打胶机发送复位请求
- **允许抓料**：当打胶机允许送料且本机就绪时，通知打胶机可以抓取边框

### 4.2 组框机信号处理

- **安全联锁**：当组框机急停或安全异常时，向组框机回传急停确认，并设置本机停止标志
- **开门请求**：当组框机请求开门时，通知组框机本机已暂停，并确认可以开门
- **就绪信号**：当本机处于自动模式、系统使能、无报警且非紧急停止状态时，向组框机发送就绪信号
- **送料请求**：当本机需要边框且就绪且上游允许时，向组框机请求输送边框

### 4.3 机器人信号处理（预留）

- **允许码料**：当本机就绪且无报警时，允许机器人进行码料
- **停止码料**：当系统故障或急停时，停止机器人码料
- **复位请求**：当本机复位且机器人有故障或急停时，向机器人发送复位请求

### 4.4 报警管理

- **报警编码**：将外部设备故障统一编码，传入公共报警FB的扩展通道
- **报警范围**：300~399段，其中：
  - 300-309：打胶机相关报警
  - 310-319：组框机相关报警
  - 320-329：机器人相关报警

### 4.5 安全条件计算

- **安全条件**：
  1. 组框机无急停
  2. 打胶机无急停且无故障
  3. 机器人无急停且无故障（预留）

## 7. 注意事项

1. **安全联锁**：外部设备的急停和故障信号会直接影响本机的安全条件，确保安全联锁逻辑正确实现
2. **通讯异常处理**：当外部设备通讯异常时，功能块会自动将关键信号置为保守值，确保系统安全
3. **机器人接口**：机器人接口目前为预留状态，待实际机器人设备接入时需要更新相应的输入/输出映射
4. **报警管理**：外部设备报警会汇总到300~399段，与本机报警区分开，便于故障定位
5. **接口一致性**：与主控程序的接口必须保持一致，确保信号传递正确
