## 2. 功能块概述

### 2.1 功能描述

外部设备交互功能块是边框缓存机系统与外部设备（组框机、打胶机、机器人）进行交互的核心组件，负责：
- 接收和处理来自外部设备的信号
- 生成并发送控制信号到外部设备
- 处理外部设备的安全联锁逻辑
- 汇总外部设备的报警信息
- 提供预留的机器人接口，支持未来功能扩展

### 2.2 设计目标

- **安全性**：确保外部设备的安全状态得到正确处理
- **可靠性**：在通讯异常时提供安全的故障处理机制
- **可扩展性**：预留接口以支持未来功能扩展
- **可维护性**：代码结构清晰，易于理解和维护

### 3.2 内部逻辑设计

#### 3.2.1 打胶机信号处理

```pascal
(* 打胶机信号有效性检查与安全回退 *)
IF i_b打胶机_通讯异常 THEN
    // 当通讯异常时, 强制将关键输入信号置为保守值(禁止状态)
    q_b允许抓料       := FALSE;     // 强制禁止抓料输出
    q_b安全区信号     := FALSE;     // 强制关闭安全区
ELSE
    // 正常情况下, 安全区信号根据系统状态控制
    q_b安全区信号     := i_b使能 AND i_b自动模式;
END_IF;

// 请求运行: 本机自动启动且打胶机未在自动运行中
IF i_b自动模式 AND i_b使能 AND NOT i_b打胶机_自动中 THEN
    q_b打胶机_请求运行 := TRUE;
ELSE
    q_b打胶机_请求运行 := FALSE;
END_IF;

// 复位请求: 本机复位且打胶机有故障或急停
IF i_b复位 AND (i_b打胶机_故障 OR i_b打胶机_急停) THEN
    q_b打胶机_复位请求 := TRUE;
ELSE
    q_b打胶机_复位请求 := FALSE;
END_IF;

// 允许抓料: 打胶机允许送料且本机就绪
IF i_b打胶机_允许送料 AND i_b本机就绪 THEN
    q_b允许抓料 := TRUE;
ELSE
    q_b允许抓料 := FALSE;
END_IF;
```

#### 3.2.2 组框机信号处理

```pascal
(* 组框机安全联锁处理 *)
// 组框机急停/安全异常 -> 本机必须立即停止 + 回传确认
IF i_b组框机_急停 OR i_b组框机_安全异常 THEN
    q_b组框机_急停 := TRUE;          // 向组框机回传急停确认
    q_b组框机紧急停止 := TRUE;         // 设置本机停止标志(传给各工站)
ELSE
    q_b组框机_急停 := FALSE;
    q_b组框机紧急停止 := FALSE;
END_IF;

// 组框机开门请求 -> 本机需暂停运行 (互锁)
IF i_b组框机_开门请求 THEN
    q_b组框机_暂停 := TRUE;          // 通知组框机本机已暂停
    q_b申请开门    := TRUE;          // 向组框机确认可以开门
ELSE
    q_b组框机_暂停 := FALSE;
    q_b申请开门    := FALSE;
END_IF;

// 就绪 = 自动模式 AND 系统使能 AND 无报警 AND 非紧急停止
q_b组框机_就绪 := i_b自动模式
                   AND i_b使能
                   AND (NOT i_b任何报警激活)
                   AND (NOT q_b组框机紧急停止);

// 当本机需要边框且就绪且上游允许时, 请求送料
IF i_b使能
   AND (NOT q_b组框机紧急停止)         // 无紧急停止
   AND i_b组框机_允许送料             // 上游允许送出
   AND i_b自动模式                  // 自动模式下才请求
THEN
    q_b组框机_请求送料 := TRUE;
ELSE
    q_b组框机_请求送料 := FALSE;
END_IF;
```

#### 3.2.3 机器人信号处理（预留）

```pascal
(* 机器人控制逻辑 (预留) *)
// 允许码料: 本机就绪且无报警
IF i_b本机就绪 AND (NOT i_b任何报警激活) THEN
    q_b机器人_允许码料 := TRUE;
ELSE
    q_b机器人_允许码料 := FALSE;
END_IF;

// 停止码料: 系统故障或急停
IF i_b系统故障 OR i_b组框机_急停 OR i_b打胶机_急停 OR i_b机器人_急停 THEN
    q_b机器人_停止码料 := TRUE;
ELSE
    q_b机器人_停止码料 := FALSE;
END_IF;

// 机器人复位请求
IF i_b复位 AND (i_b机器人_故障 OR i_b机器人_急停) THEN
    q_b机器人_复位请求 := TRUE;
ELSE
    q_b机器人_复位请求 := FALSE;
END_IF;
```

#### 3.2.4 报警管理

```pascal
(* 外部设备报警码汇总 *)
// 将外部设备故障统一编码, 传入公共报警FB的扩展通道
IF i_b打胶机_故障 THEN
    i外部设备报警码 := 300;        // 300=打胶机设备故障
ELSIF i_b打胶机_急停 THEN
    i外部设备报警码 := 301;        // 301=打胶机急停
ELSIF i_b打胶机_通讯异常 THEN
    i外部设备报警码 := 302;        // 302=打胶机通讯异常
ELSIF i_b组框机_急停 THEN
    i外部设备报警码 := 310;        // 310=组框机急停
ELSIF i_b组框机_安全异常 THEN
    i外部设备报警码 := 311;        // 311=组框机安全系统异常
ELSIF i_b组框机_通讯异常 THEN
    i外部设备报警码 := 312;        // 312=组框机通讯异常
ELSIF i_b机器人_故障 THEN
    i外部设备报警码 := 320;        // 320=机器人设备故障
ELSIF i_b机器人_急停 THEN
    i外部设备报警码 := 321;        // 321=机器人急停
ELSIF i_b机器人_通讯异常 THEN
    i外部设备报警码 := 322;        // 322=机器人通讯异常
ELSE
    i外部设备报警码 := 0;           // 0=无外部设备报警
END_IF;

q_i外部设备报警汇总 := i外部设备报警码;
q_b外部设备故障 := (i外部设备报警码 <> 0);
```

#### 3.2.5 安全条件计算

```pascal
(* 安全条件计算 *)
(* 安全条件检查:
   1. 组框机无急停
   2. 打胶机无急停且无故障
   3. 机器人无急停且无故障 (预留)
*)
IF NOT i_b组框机_急停              // 组框机急停联锁
   AND NOT i_b打胶机_急停           // 打胶机急停联锁
   AND NOT i_b打胶机_故障            // 打胶机故障联锁
   AND NOT i_b机器人_急停            // 机器人急停联锁(预留)
   AND NOT i_b机器人_故障            // 机器人故障联锁(预留)
THEN
    b安全条件满足 := TRUE;
ELSE
    b安全条件满足 := FALSE;
END_IF;

q_b系统安全条件满足 := b安全条件满足;
```

### 4.2 错误处理

#### 4.2.1 通讯异常处理

当检测到外部设备通讯异常时，功能块会自动将相关输出信号置为安全状态：
- 打胶机通讯异常：强制禁止抓料，关闭安全区
- 组框机通讯异常：视为安全异常，触发急停联锁
- 机器人通讯异常：视为安全异常，触发急停联锁

#### 4.2.2 故障处理

当检测到外部设备故障时：
- 生成相应的报警码
- 触发安全联锁逻辑
- 等待操作人员处理故障并复位
