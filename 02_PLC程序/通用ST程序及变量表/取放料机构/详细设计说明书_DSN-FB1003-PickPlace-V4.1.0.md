## 3. 功能描述

### 3.1 核心功能

FB_1003_PickPlace_BufferFraming 是本项目**最复杂的功能块**，实现边框缓存机取放料机构的完整控制逻辑：

- **11步自动状态机**: 完整的取放料循环控制（空闲 → 等待放料完成 → 取料下降 → 夹紧 → 检测 → 上升 → 移动 → 放料下降 → 松开 → 上升 → 完成通知）
- **4层循环管理**: 一次取两根边框（前后夹紧同时动作），4层总共循环取2次完成全部取料
- **自动/手动双模式**: 支持自动循环运行和手动点动调试（14个手动操作点）
- **初始化流程**: 上电或复位后自动执行气缸复位和伺服使能初始化
- **内置报警机制**: 8种报警类型的检测、锁存与输出（101~108段）
- **双夹爪同时动作**: 前后夹紧同时夹紧/松开，一次抓取两根边框
- **产品检测逻辑**: 4个光电传感器AND判断，确保取料可靠
- **完全解耦设计**: 无任何X/Y/M/D物理地址引用，所有数据通过逻辑变量交互

### 3.2 核心工艺说明

```
核心工艺: 一次取两根边框, 4层总共循环取2次

取料顺序:
┌─────────────────────────────────────────────────────┐
│  第1次取料 (s_i已完成层数=0)                        │
│  ├── 等待 L1+L2 放料完成                            │
│  ├── Z轴下降 -> 前后夹紧同时夹紧(一次取两根)        │
│  ├── 产品检测(长边x2 + 短边x2, 4传感器AND判断)      │
│  ├── Z轴上升 -> X1轴移动到放料位置                  │
│  ├── Z轴下降 -> 前后夹紧同时松开                    │
│  ├── Z轴上升 -> 通知下游(打胶机送料机构)            │
│  └── s_i已完成层数 += 2 (变为2)                     │
│                                                     │
│  第2次取料 (s_i已完成层数=2)                        │
│  ├── 等待 L3+L4 放料完成                            │
│  ├── 重复上述取放料流程                             │
│  └── s_i已完成层数 += 2 (变为4, 全部完成)           │
│                                                     │
│  循环重置: s_i已完成层数 >= 4 时归零, 等待下一轮启动 │
└─────────────────────────────────────────────────────┘
```

## 5. 状态机设计

### 5.1 11步状态转换总图

```
FB_1003 PickPlace 11步状态机:

步骤0(空闲) → 步骤1(等待放料完成) → 步骤2(Z轴下降) → 步骤3(前后夹紧)
→ 步骤4(产品检测) → 步骤5(Z轴上升) → 步骤6(X1轴移动到放料位)
→ 步骤7(Z轴下降) → 步骤8(前后松开) → 步骤9(Z轴上升)
→ 步骤10(发送完成通知) → 回到步骤1

循环特点: 
- 4层循环: 第1次取L1+L2, 第2次取L3+L4, 共2次完成全部
- 双夹爪: 前后夹紧同时动作, 一次取两根边框
- 安全互锁: 与四层输送机和打胶机送料机构的时序配合
```

### 5.2 各步骤详细说明表

| 步骤 | 名称 | 输入条件 | 动作输出 | 转出条件 | 超时报警 | 典型时间 |
|:----:|:-----|:---------|:---------|:---------|:-------:|:-------:|
| 0 | 空闲/待机 | - | 全部OFF | 启动触发 | - | - |
| 1 | 等待放料完成 | 启动 | 无 | L1+L2或L3+L4放料完成 | - | 取决于上游 |
| 2 | Z轴下降 | 放料完成 | Z轴下降使能 | 下限位/到位 | ALM_103 | ~1~2s |
| 3 | 前后夹紧 | 到位 | 前/后夹紧电磁阀ON | 动点(夹紧位) | ALM_105/106 | ~0.5s |
| 4 | 产品检测 | 夹紧完成 | 无 | 4传感器AND | ALM_107 | 即时 |
| 5 | Z轴上升 | 检测通过 | Z轴上升 | 原点(上位) | ALM_104 | ~1~2s |
| 6 | X1轴移动 | 上升完成 | X1轴请求移动 | 目标位到达 | ALM_108 | ~2~3s |
| 7 | Z轴下降 | X1到位 | Z轴下降 | 下限位 | ALM_103 | ~1~2s |
| 8 | 前后松开 | 到位 | 前/后夹紧OFF | 原点(松开位) | ALM_105/106 | ~0.5s |
| 9 | Z轴上升 | 松开完成 | Z轴上升 | 原点(上位) | ALM_104 | ~1~2s |
| 10 | 完成通知 | 上升完成 | o_b放料完成=TRUE | 脉冲结束 | - | 0.2s |

## 6. 报警机制设计

### 6.1 报警码定义 (101~108段)

| 报警码 | 含义 | 触发条件 | 触发步骤 | 清除条件 | 优先级 |
|--------|------|----------|----------|----------|:------:|
| 101 | Z轴伺服故障 | Z轴驱动器ALM信号 | - | 故障消除+复位 | 高 |
| 102 | X1轴伺服故障 | X1轴驱动器ALM信号 | - | 故障消除+复位 | 高 |
| 103 | 升降下降超时 | Z轴未在规定时间内到达下位 | 步骤2/7 | 复位按钮 | 中 |
| 104 | 上升超时 | Z轴未在规定时间内回到上位 | 步骤5/9 | 复位按钮 | 中 |
| 105 | 前夹紧超时 | 前夹紧未在规定时间内夹紧到位 | 步骤3 | 复位按钮 | 中 |
| 106 | 后夹紧超时 | 后夹紧未在规定时间内夹紧到位 | 步骤3 | 复位按钮 | 中 |
| 107 | 产品检测失败 | 4个产品传感器未全部检测到 | 步骤4 | 排除原因+复位 | 低 |
| 108 | 移动超时 | X1轴未在规定时间内到达目标位 | 步骤6 | 复位按钮 | 中 |

## 7. 初始化流程

```
初始化触发条件:
  1. 首次使能(i_b使能 FALSE->TRUE)
  2. 复位操作(i_b复位=TRUE)
  3. 故障清除后

初始化执行内容:
  ├─ 所有输出归零(气缸电磁阀OFF, 伺服使能OFF)
  ├─ s_i步序 := STP_空闲(0)
  ├─ s_i已完成层数 := 0
  └─ 等待各气缸回到安全位置(超时容错10秒)
```

## 附录: 特殊设计决策记录

| 决策ID | 决策内容 | 原因 | 替代方案(未采用) |
|:------:|:---------|:-----|:---------------|
| D001 | 采用11步而非简化为更少步骤 | 保证每步职责单一，便于调试和维护 | 合并某些步骤(降低可读性) |
| D002 | 4层循环取2次而非一次性取4层 | 降低单次取料的复杂度和风险 | 单次取4层(机械复杂度太高) |
| D003 | 前后夹紧同时动作而非顺序动作 | 提高效率，一次取两根 | 顺序夹紧(效率减半) |
| D004 | 4传感器AND判断 | 提高取料可靠性，防止单传感器误报 | 单传感器或OR判断(可靠性低) |
| D005 | 使用内部计数器(s_i已完成层数) | 简化4层循环逻辑，无需外部状态管理 | 外部状态变量(增加耦合度) |
