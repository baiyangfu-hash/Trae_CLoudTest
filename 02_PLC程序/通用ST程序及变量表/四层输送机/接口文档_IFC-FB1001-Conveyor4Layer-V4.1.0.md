## 3. FB_1002_SingleLayerConveyor 输入接口定义 (VAR_INPUT)

> 以下为单层基础单元的完整24个输入接口定义。在容器模式下，这些输入通过 `FB_1001_Conveyor4Layer` 的公共信号和分层ARRAY信号映射到各层实例。

### 3.1 系统控制信号组 (3个BOOL)

#### 3.1.1 i_b使能

| 属性 | 值 |
|------|-----|
| **针脚名称** | `i_b使能` |
| **数据类型** | BOOL |
| **功能描述** | 总使能信号，控制本功能块的运行/停止 |
| **有效值域** | TRUE=允许运行, FALSE=停止并清零全部输出 |
| **默认行为** | FALSE时强制归零所有输出，状态机和计数器复位，定时器停止 |
| **来源** | 主控系统就绪标志（系统急停解除、安全门关闭等条件满足后置TRUE） |
| **更新周期** | 每扫描周期 |
| **优先级** | 最高（覆盖所有其他输入） |

#### 3.1.2 i_b自动模式

| 属性 | 值 |
|------|-----|
| **针脚名称** | `i_b自动模式` |
| **数据类型** | BOOL |
| **功能描述** | 自动运行模式选择信号 |
| **有效值域** | TRUE=选中自动模式, FALSE=未选中 |
| **互斥关系** | 与`i_b手动模式`互斥（不能同时为TRUE） |
| **来源** | HMI M区变量（操作员通过HMI界面切换） |

#### 3.1.3 i_b手动模式

| 属性 | 值 |
|------|-----|
| **针脚名称** | `i_b手动模式` |
| **数据类型** | BOOL |
| **功能描述** | 手动调试模式选择信号 |
| **有效值域** | TRUE=选手动模式, FALSE=未选中 |
| **互斥关系** | 与`i_b自动模式`互斥（不能同时为TRUE） |
| **来源** | HMI M区变量（操作员通过HMI界面切换） |

### 3.2 操作按钮组 (3个BOOL)

#### 3.2.1 i_b启动

| 属性 | 值 |
|------|-----|
| **针脚名称** | `i_b启动` |
| **数据类型** | BOOL |
| **功能描述** | 自动循环启动按钮信号 |
| **触发方式** | **上升沿触发**（建议主控做边沿检测） |
| **生效条件** | 仅在`i_b自动模式=TRUE`且`i_b使能=TRUE`时有效 |
| **来源** | HMI按钮 |

#### 3.2.2 i_b停止

| 属性 | 值 |
|------|-----|
| **针脚名称** | `i_b停止` |
| **数据类型** | BOOL |
| **功能描述** | 自动循环停止按钮信号 |
| **触发方式** | **电平有效**（按下即停止） |
| **生效条件** | 无需模式判断，随时可停止 |
| **行为描述** | 按下后状态机立即回到步骤0(空闲)，清除所有运动请求 |
| **来源** | HMI按钮 |

#### 3.2.3 i_b复位

| 属性 | 值 |
|------|-----|
| **针脚名称** | `i_b复位` |
| **数据类型** | BOOL |
| **功能描述** | 故障复位/初始化按钮信号 |
| **触发方式** | **电平有效**（按下执行复位） |
| **行为描述** | 清除所有报警锁存标志，重新执行初始化流程 |
| **前置条件** | 必须先排除故障原因（如传感器故障、超时条件等） |
| **来源** | HMI按钮 |

### 3.3 手动操作信号组 (6个BOOL)

> 包括: i_b阻挡下降/上升, i_b分料推出/复位, i_b输送正转/反转
> 详细定义参见详细设计说明书_DSN-FB1001-Conveyor4Layer-V4.1.0.md 第8章

### 3.4 工艺参数组 (REALx1 + INTx2)

> 包括: i_r输送速度, i_i分料时间, i_i阻挡等待时间
> 默认值回退机制详见DSN文档第9章

### 3.5 传感器输入组 (7个BOOL)

> 包括: i_分料前感应器, i_到位感应器1/2, i_阻挡气缸上位/下位, i_分料气缸上位/下位
> 全部来自主控IO映射，本功能块不直接读取X地址

### 3.6 上下游联锁信号 (1个BOOL)

#### 3.6.1 i_b上游允许放料

| 属性 | 值 |
|------|-----|
| **针脚名称** | `i_b上游允许放料` |
| **数据类型** | BOOL |
| **功能描述** | 上游设备已取走上次物料，允许本层继续放下一批 |
| **信号来源** | 取放料机构/打胶机的完成反馈信号 |
| **有效电平** | TRUE = 上游已完成取料，可以继续放料 |

## 4. FB_1002_SingleLayerConveyor 输出接口定义 (VAR_OUTPUT)

### 4.1 执行器请求输出组 (5个BOOL)

> 这些输出需要由主控程序映射到物理Y地址。
> 包括: o_阻挡电磁阀, o_分料电磁阀, o_输送带正转/慢速/反转

### 4.2 下游信号输出组 (1个BOOL)

#### 4.2.1 o_b放料完成

| 属性 | 值 |
|------|-----|
| **针脚名称** | `o_b放料完成` |
| **数据类型** | BOOL |
| **功能描述** | 该层放料完成通知脉冲信号 |
| **接收方** | 取放料机构/打胶机（作为"可以来取料"的通知） |
| **时序特性** | 脉冲信号，宽度约500ms |
| **激活时机** | 仅在步骤8（放料完成）期间为TRUE |

### 4.3 状态/HMI显示输出组 (3个)

> 包括: o_b运行中, o_b故障, o_i当前状态(INT, 0~8正常, 99故障)

### 4.4 报警输出组 (1个INT)

#### 4.4.1 o_i报警代码

| 属性 | 值 |
|------|-----|
| **针脚名称** | `o_i报警代码` |
| **数据类型** | INT |
| **功能描述** | 当前有效的报警代码（0=无报警 / 1~7有报警） |
| **接收方** | 公共报警管理器 (FB_2001_CommonAlarm_AllStation) |
| **优先级规则** | 数值越小优先级越高；多个报警同时存在时输出最小值 |

**完整报警码定义表 (1~7段)**:

| 报警码 | 名称 | 触发条件 | 触发步骤 | 清除条件 | 优先级 |
|:------:|:-----|:---------|:--------:|:---------|:------:|
| 0 | 无报警 | - | - | - | - |
| 1 | 阻挡下降超时 | 阻挡气缸未在规定时间内到达下位 | 步骤2 | 复位按钮 | 高 |
| 2 | 阻挡上升超时 | 阻挡气缸未在规定时间内到达上位 | 步骤6 | 复位按钮 | 高 |
| 3 | 分料推出超时 | 分料气缸未在规定时间内到达下位 | 步骤7 | 复位按钮 | 中 |
| 4 | 复位超时 | 故障复位后设备未能初始化到安全状态 | 初始化 | 复位按钮 | 高 |
| 5 | 输送超时 | 材料未在规定时间内到达/离开到位位置 | 步骤1/步骤3 | 复位按钮 | 高 |
| 6 | 传感器冲突 | 同一气缸上下位传感器同时为TRUE | 持续检测 | 冲突消失后自动清除 | 低 |
| 7 | 变频器故障 | 变频器反馈故障信号(ALM) | - | 故障消除+复位 | **最高** |

### A.2 报警码速查 (1~7段)

```
报警代码速查 (四层输送机专用):

  0   = 无报警 (正常)
  1   = 阻挡下降超时 (步骤2)
  2   = 阻挡上升超时 (步骤6)
  3   = 分料推出超时 (步骤7)
  4   = 复位超时 (初始化)
  5   = 输送超时      (步骤1/3)
  6   = 传感器冲突    (上下位同时TRUE)
  7   = 变频器故障    (最高优先级!)

处理方法:
  1~5,7 → 排除原因 → 按复位按钮
  6     → 检查磁性开关间距(推荐>10mm)
```
