## 2. 前置条件与依赖项

### 2.1 硬件环境要求

| 项目 | 要求 | 说明 |
|------|:----:|:-----|
| **PLC型号** | 支持ST语言的控制器（如西门子S7-1200/1500、三菱FX5U等） | 必须支持结构化文本编程和ARRAY数据类型 |
| **DI模块** | DI3 + DI4（各16点输入模块） | 共28个DI点用于4层传感器(7x4) |
| **DO模块** | DO1 + DO2（各16点输出模块） | 共20个DO点用于4层执行器(5x4) |
| **气缸系统** | 阻挡气缸x4 + 分料气缸x8（每层各1个） | 每个气缸配2个磁性开关(上位/下位) |
| **输送带/变频器** | 输送带x4 + 变频器x4(VF1~VF4) | 每台变频器支持正转/慢速/反转3路DI控制 |
| **传感器** | 光电传感器x14(分料前+到位x2)x4层 | 对射式或漫反射式，需调整灵敏度 |
| **HMI界面** | 已配置操作按钮、状态显示、参数设定画面 | 启动/停止/复位、手动操作按钮、步序显示 |

### 2.2 软件依赖

| 依赖项 | 版本要求 | 用途 |
|--------|:-------:|:-----|
| 主控程序 (PRG_MainControl_DJ2026005) | V4.1.0+ | IO映射和调用本FB容器 |
| 公共报警FB (FB_2001_CommonAlarm_AllStation) | V4.1.0+ | 接收1~7段报警代码 |
| 取放料机构FB (FB_1003_PickPlace_BufferFraming) | V4.1.0+ | 提供`i_b取放料完成[1..4]`上游信号 |

### 2.3 必须连接的信号清单（最小化调用）

| 针脚名称 | 连接建议 | 原因 |
|---------|:---------|:-----|
| `i_b使能` | `gb系统就绪` | 总开关，FALSE时全部输出归零 |

## 3. 数组索引约定

```
数组索引约定:

┌─────────────────────────────────────────────────────┐
│                  物理层级布局                       │
│                                                     │
│    ┌─────────┐  L4 = 第4层 = 顶层  = 数组索引 [4]   │
│    │         │                                     │
│    ├─────────┤  L3 = 第3层        = 数组索引 [3]     │
│    │         │                                     │
│    ├─────────┤  L2 = 第2层        = 数组索引 [2]     │
│    │         │                                     │
│    └─────────┘  L1 = 第1层 = 底层  = 数组索引 [1]   │
│                                                     │
│  材料流向: 上方进料 → 下方出料(给取放料机构)        │
│                                                     │
└─────────────────────────────────────────────────────┘

IO模块对应关系:
  L4(顶层) ←→ DI3:X0~X7  + DO1:Y0~Y4   (第1组)
  L3       ←→ DI3:X11~X17 + DO1:Y10~Y14  (第2组)
  L2       ←→ DI4:X1~X7  + DO2:Y0~Y4    (第3组)
  L1(底层) ←→ DI4:X11~X17 + DO2:Y10~Y14  (第4组)
```

## 4. ST调用示例

### 4.1 容器模式完整调用示例 (推荐)

> 以下为在主控程序中通过 `FB_1001_Conveyor4Layer` 容器管理4层输送机的完整ST代码。
> 详细代码请参考旧版本文档或DSN文档中的架构图。

```st
(*==============================================================================
  主控程序 (PRG_MainControl_DJ2026005_V4.1.0) 中调用 FB_1001_Conveyor4Layer 的完整示例
  项目: DJ-2026-005 边框缓存机
  版本: V4.1.0
  模式: 容器模式 (推荐, 统一管理4层)
  ==============================================================================*)

PROGRAM PRG_MainControl_DJ2026005
VAR
    (*--- 四层输送机容器实例 ---*)
    fb四层输送机 : FB_1001_Conveyor4Layer_BufferFraming;
    
    (*--- 边沿检测 ---*)
    rtrig_Start : R_TRIG;  (* 启动按钮上升沿检测 *)
    
    (*--- 中间变量: 用于IO映射 ---*)
    (* 详见旧版本文档第4章完整代码 *)
END_VAR


(*==============================================================================
  调用四层输送机容器功能块 (核心部分)
  ==============================================================================*)
fb四层输送机(
    (*=== 输入连接 (INPUT) ===*)
    
    (* 一、系统控制信号 (公共, 所有层共享) *)
    i_b使能               := gb系统就绪,
    i_b自动模式           := gm自动模式选择,
    i_b手动模式           := gm手动模式选择,
    
    (* 二、操作按钮 (公共) *)
    i_b启动               := rtrig_Start.Q,    (* 上升沿! *)
    i_b停止               := gm停止按钮,
    i_b复位               := gm复位按钮,
    
    (* 三、工艺参数 (公共) *)
    i_r输送速度           := gr输送速度设定,
    i_i分料时间           := gi分料时间设定,
    i_i阻挡等待时间        := gi阻挡等待时间设定,
    
    (* 四、手动操作信号 (每层独立, ARRAY[1..4]) *)
    i_b阻挡下降[1]        := gmL1阻挡下降,
    (* ... 其他23个分层信号 ... *)
    
    (* 五、传感器输入 (每层独立, 来自IO映射) *)
    i_分料前感应器[4]      := diL4分料前感应器,   (* DI3:X0 *)
    (* ... 其他27个传感器信号 ... *)
    
    (* 六、下游反馈信号 (每层独立) *)
    i_b取放料完成[1]       := giL1取放料完成,
    (* ... 其他3层 ... *)
    
    
    (*=== 输出连接 (OUTPUT) ===*)
    
    (* 一、执行器请求输出 -> 映射到物理DO地址 *)
    o_阻挡电磁阀[4]        => doL4阻挡电磁阀,    (* DO1:Y0  -> YV1 *)
    (* ... 其他19个执行器输出 ... *)
    
    (* 二、下游信号输出 -> 给取放料机构 *)
    o_b放料完成[4]         => goL4放料完成,
    (* ... 其他3层 ... *)
    
    (* 三、状态/HMI显示输出 *)
    o_b运行中              => go四层运行中,
    o_b故障                => go四层故障,
    o_i当前状态            => gi四层当前状态,
    o_Lx当前步序[4]        => giL4当前步序,
    (* ... 其他步序 ... *)
    
    (* 四、报警输出 -> 给公共报警管理器 *)
    o_i本站报警代码        => gi四层报警码
);
```

## 5. 参数调优建议

### 5.1 关键参数推荐值

| 参数名 | 默认值 | 推荐范围 | 调优依据 | 影响因素 |
|--------|:------:|:--------:|:---------|:---------|
| `i_i阻挡等待时间` | 800 ms | 500 ~ 1500 ms | 气缸行程长度、气压压力(0.4~0.6MPa) | 过短导致误报ALM_1/2，过长影响节拍 |
| `i_i分料时间` | 1000 ms | 800 ~ 2000 ms | 分料气缸行程、负载重量 | 根据边框尺寸调整 |
| `i_r输送速度` | 50.0 Hz | 30 ~ 80 Hz | 变频器额定频率、电机极数 | 超过额定值会损坏电机 |
| `T_DEF_输送超时时间` | 5000 ms | 3000 ~ 8000 ms | 输送带长度、材料重量 | 初次调试建议加大到8000ms |
| `T_DEF_慢速送出时间` | 2000 ms | 1500 ~ 3000 ms | 慢速距离、定位精度要求 | 影响取料位置精度 |
| `T_DEF_到位稳定时间` | 500 ms | 300 ~ 800 ms | 传感器响应速度、机械振动 | 影响步骤3->4的转换稳定性 |

### 5.2 典型工况参数配置参考

| 工况描述 | 阻挡等待(ms) | 分料时间(ms) | 输送速度(Hz) | 输送超时(ms) | 备注 |
|---------|:-----------:|:----------:|:----------:|:----------:|:-----|
| 标准边框(小尺寸) | 800 | 1000 | 50.0 | 5000 | 默认配置 |
| 大尺寸边框 | 1200 | 1500 | 40.0 | 8000 | 增加时间和降低速度 |
| 高速生产模式 | 600 | 800 | 70.0 | 4000 | 设备状态良好时使用 |
| 低温环境(<10°C) | 1000 | 1200 | 45.0 | 6000 | 气缸动作变慢，增加余量 |
| 调试/维护模式 | 1500 | 2000 | 30.0 | 10000 | 最大安全余量 |

## 6. 多层联动测试步骤

### 6.1 分阶段调试流程

```
调试阶段划分:

阶段1: 手动验证 [预计40分钟]
│  ├─ Step 1.1: 确认HMI选手动模式，逐层测试6个手动操作按钮(共24个)
│  ├─ Step 1.2: 验证每个气缸的动作方向正确(上升/下降/推出/复位)
│  ├─ Step 1.3: 验证输送带正转/反转/慢速方向正确(共12个方向)
│  ├─ Step 1.4: 确认所有传感器信号正常(无冲突、无抖动,共28个)
│  └─ Step 1.5: 验证同层内互锁关系(如阻挡上下互斥、输送正反互斥)
│
阶段2: 单层自动测试 [预计60分钟]
│  ├─ Step 2.1: 只启用L1(底层), 其他3层保持无材料状态
│  ├─ Step 2.2: 放入测试材料, 按启动观察9步状态机流转
│  ├─ Step 2.3: 记录每步耗时, 确认均在超时保护范围内
│  ├─ Step 2.4: 人为制造超时条件, 验证报警触发和复位功能
│  └─ Step 2.5: 依次对L2/L3/L4重复上述测试, 确保每层独立工作正常
│
阶段3: 多层联动测试 [预计60分钟]
│  ├─ Step 3.1: 4层同时放入材料, 观察是否独立运行互不干扰
│  ├─ Step 3.2: 验证某层故障时不影响其他层运行(隔离性)
│  ├─ Step 3.3: 验证停止按钮可同时停止所有层
│  ├─ Step 3.4: 验证汇总状态输出(o_b运行中/o_b故障/o_i当前状态)正确
│  ├─ Step 3.5: 验证o_Lx当前步序[1..4]各自独立反映各层状态
│  └─ Step 3.6: 验证多报警时优先级选择(o_i本站报警代码取MIN)
│
阶段4: 上游联调 [预计90分钟]
│  ├─ Step 4.1: 连接取放料机构, 验证o_b放料完成[1..4]信号时序(~500ms脉冲)
│  ├─ Step 4.2: 验证i_b取放料完成[1..4]信号的联锁逻辑
│  ├─ Step 4.3: 测试循环连续性(一层完成后能否自动开始下一轮)
│  ├─ Step 4.4: 全流程连续运行24小时稳定性测试
│  └─ Step 4.5: 断电重启后初始化流程验证
```

## 7. 常见问题排查指南

### 7.1 问题诊断表

| 序号 | 现象 | 可能原因 | 排查方法 | 解决方法 | 相关报警 |
|:----:|:-----|:---------|:---------|:---------|:--------|
| 1 | 按启动后无反应 | ① `i_b使能`=FALSE<br>② 未选中自动模式<br>③ 分料前感应器未检测到材料<br>④ `i_b上游允许放料`=FALSE<br>⑤ 启动信号未做边沿检测 | ① 检查系统就绪条件<br>② 切换到自动模式<br>③ 监控`i_分料前感应器`<br>④ 检查取放料机构状态<br>⑤ 使用R_TRIG检测上升沿 | 排除对应原因 | - |
| 2 | 阻挡下降后立即报警 | ① 气缸动作太慢<br>② 下位传感器松动/接线不良<br>③ 气压过低(<0.4MPa)<br>④ `i_i阻挡等待时间`设置过短 | 监控`i_阻挡气缸下位`信号是否到达；测量实际动作时间 | 加大`i_i阻挡等待时间`(建议1200ms)；检修传感器；调节气压 | ALM_1 |
| 3 | 输送带运转但始终不到位 | ① 输送带卡死/打滑<br>② 到位传感器损坏或偏移<br>③ 材料尺寸异常过大<br>④ 双传感器中某一个始终OFF | 监控双到位传感器信号是否同时为TRUE | 检查机械传动、皮带张紧度；调整/更换传感器；检查物料规格 | ALM_5 |
| 4 | 传感器冲突报警(码6) | ① 磁性开关间距太小同时感应<br>② 传感器安装松动导致震动误触发<br>③ 气缸未完全到位但两开关都触发 | 监控`i_阻挡气缸上位 AND i_阻挡气缸下位`是否同时TRUE | 加大磁性开关安装间距(推荐>10mm)；紧固传感器安装 | ALM_6 |
| 5 | 手动操作按钮无效 | ① 未选手动模式<br>② 该层存在故障<br>③ HMI按钮变量映射错误 | 检查`i_b手动模式`和`o_b故障`状态 | 切换到手动模式；先按复位排除故障；检查HMI变量地址 | - |
| 6 | 放料完成信号不发 | ① 步骤7的分料复位验证失败<br>② 分料气缸复位行程不足<br>③ `T_DEF_慢速送出时间`过短 | 监控步骤7中`i_分料气缸上位`信号 | 检查分料气缸复位是否到达上位；加大分料气缸行程；增加慢速送出时间 | ALM_3 |

  ALM_3 分料推出超时 → 检查分料气缸/下位传感器/加大分料时间
  ALM_4 复位超时      → 检查所有气缸安全位置/延长初始化超时
  ALM_5 输送超时     → 检查皮带/变频器/到位传感器
  ALM_6 传感器冲突   → 加大磁性开关间距(>10mm), 自动恢复
  ALM_7 变频器故障   → 最高优先! 检查变频器

  1. 看报警码 → 确定类型
  2. 看o_Lx当前步序 → 确定哪层+哪步
  3. 排除物理原因
  4. 按复位按钮
  5. 若反复出现 → 调整参数或检修硬件
