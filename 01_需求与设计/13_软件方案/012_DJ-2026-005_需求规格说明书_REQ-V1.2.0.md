### 4.2 流程需求

#### 4.2.1 系统整体流程

系统采用**三工站流水线架构**，各工站通过放料完成信号串联：

```mermaid
graph LR
    A[四层输送机<br/>FB_1001] -->|L1~L4放料完成| B[取放料机构<br/>FB_1003]
    B -->|放料完成通知| C[打胶机送料机构<br/>FB_1004]
    C -->|X2轴返回待机| A
```

#### 4.2.2 四层输送机自动流程（每层独立9步状态机）

| 步序 | 状态名称 | 动作描述 | 执行器输出 | 传感器条件 |
| ---- | -------- | -------- | ---------- | ---------- |
| 0 | STP_空闲 | 空闲待机，等待启动信号 | 无 | — |
| 1 | STP_检测材料到位 | 等待分料前感应器检测到边框材料 | 无 | 分料前感应器=ON |
| 2 | STP_阻挡下降 | 阻挡气缸下降，阻止后续材料进入 | 阻挡电磁阀=ON(下降) | 阻挡气缸下位=ON |
| 3 | STP_输送正转 | 输送带正向运转，将材料送到分料位置 | 输送带正转=ON | — |
| 4 | STP_到位检测 | 双传感器冗余检测材料到位 | 输送带停止 | 到位感应器1 AND 到位感应器2 = ON |
| 5 | STP_阻挡上升 | 阻挡气缸上升释放，允许后续材料进入 | 阻挡电磁阀=OFF(上升) | 阻挡气缸上位=ON |
| 6 | STP_分料动作 | 分料气缸推出，将边框分离到取料位置 | 分料电磁阀=ON(推出) | 分料气缸下位=ON |
| 7 | STP_慢速送出 | 输送带慢速精确送出到取料位 | 输送带慢速=ON | — |
| 8 | STP_放料完成 | 放料完成，等待取放料机构取走 | 所有执行器复位 | 取放料完成后进入下一循环 |

#### 4.2.3 取放料机构自动流程（11步状态机）

| 步序 | 状态名称 | 动作描述 | 关键细节 |
| ---- | -------- | -------- | -------- |
| 0 | STP_空闲 | 空闲待机 | — |
| 1 | STP_等待放料完成 | 等待任意一层输送机放料完成信号 | 接收L1~L4放料完成信号 |
| 2 | STP_取料下降 | Z轴下降到取料高度 | Z轴伺服定位 |
| 3 | STP_夹爪夹紧 | 前/后夹紧气缸**同时夹紧**，一次取**两根边框** | 前夹紧+后夹紧同时动作，含夹紧2（第二组夹爪） |
| 4 | STP_产品检测 | 4个光电传感器检测边框存在 | 长边1/长边2/短边1/短边2 全部=ON |
| 5 | STP_取料上升 | Z轴上升到安全高度 | 避免与输送机干涉 |
| 6 | STP_移动到放料位置 | X1轴横向移动到打胶机侧放料位置 | X1轴伺服定位 |
| 7 | STP_放料下降 | Z轴下降到放料高度 | Z轴伺服定位 |
| 8 | STP_夹爪松开 | 前/后夹紧同时松开 | 释放两根边框 |
| 9 | STP_放料上升 | Z轴上升到安全高度 | — |
| 10 | STP_放料完成通知 | 发送放料完成信号给下游打胶机送料机构 | 触发打胶机送料FB开始工作 |

#### 4.2.4 打胶机送料机构自动流程（6步状态机）

| 步序 | 状态名称 | 动作描述 | 外部交互 |
| ---- | -------- | -------- | -------- |
| 0 | STP_空闲 | 空闲待机 | — |
| 1 | STP_等待取放料完成 | 等待取放料机构放料完成信号 | 接收取放料_放料完成 |
| 2 | STP_移动到取料位置 | X2轴移动到取料位置（取放料机构侧） | X2轴伺服定位 |
| 3 | STP_等待打胶机允许 | 等待打胶机发送允许送料信号 | 接收X76:打胶机允许送料 |
| 4 | STP_移动到打胶机位置 | X2轴移动到打胶机抓取位置 | 输出Y47:安全区信号(关闭保护) |
| 5 | STP_等待打胶机取料 | 等待打胶机完成边框抓取 | 接收X102:打胶机取料完成，输出Y44:允许抓料 |
| 6 | STP_返回待机位置 | X2轴返回初始待机位置 | 输出Y47:安全区信号(开放) |

**架构特点**：
- **纯协调层主控**：主控程序不包含任何业务逻辑，仅做IO映射和工站调度
- **纯逻辑块工站**：所有工站FB无物理地址依赖，完全通过接口变量与主控交换数据
- **集中式IO映射**：所有X/Y地址映射集中在主控程序的Step1(输入)和Step8(输出)
- **严格执行顺序**：输入映射→HMI读取→外部信号→工站调用(4→5→6→7)→输出映射→HMI回写→D区回写

### 8.2 数据安全

- 重要参数设置需要密码保护（HMI权限分级）
- 报警记录不可修改（只读D区）
- 安全相关信号（急停/安全门）采用硬件常闭触点，软件仅读取状态
- D区报警数据固定分配，不随程序修改而变动
