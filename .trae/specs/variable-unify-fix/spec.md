# PLC变量命名全面统一修复 Spec (V2)

## Why
主控程序中存在**变量命名不一致**的严重问题：部分变量带`i_b`/`q_b`前缀，部分仍为旧的`b`开头格式。同时**文档和变量表中仍引用旧名**，导致代码与文档不同步。必须一次性全部统一。

## What Changes
- **统一所有输入变量为 `i_` 前缀格式**
- **统一所有输出变量为 `q_` 前缀格式**
- **同步更新所有关联文档和变量表中的旧名引用**

- 受影响代码文件: [主控程序_V4.0.0.st](通用ST程序及变量表/主控/主控程序_V4.0.0.st)
