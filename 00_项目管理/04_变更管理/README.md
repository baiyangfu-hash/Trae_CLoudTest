## 变更管理原则

### Single Source of Truth原则
每个变更的唯一权威数据源是其变更单文件，台帐只做索引和统计。

### 引用模式
```
台帐(索引/统计) ──超链接──▶ 变更单(完整档案)
     ↑                           │
  §4 一行摘要               §1~§10 全部细节
  §3 多维统计               §6 影响评估
  §5 趋势分析               §8 审批记录
                           §10 验证结论
```
