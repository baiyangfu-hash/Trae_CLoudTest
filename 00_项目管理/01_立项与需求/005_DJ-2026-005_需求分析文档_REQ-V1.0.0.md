## 3. 项目概述
边框缓存机是一种用于自动化生产线的设备，主要功能包括边框材料的缓存、分料和送料。设备通过PLC控制伺服电机实现精确的位置控制，通过HMI界面实现操作和监控。

**重要说明**：本设备是配合打胶机使用的专用设备，负责为打胶机提供边框材料的缓存和送料功能。

## 3. 功能需求

### 3.1 核心功能
| 功能模块 | 功能描述 | 技术实现 |
|----------|----------|----------|
| 边框缓存 | 存储和管理边框材料 | PLC控制 + 传感器检测 |
| 分料功能 | 自动分离不同规格的边框 | 伺服轴运动控制 |
| 送料功能 | 将边框材料送到指定位置 | 多轴协调控制 |
| 状态监控 | 实时监控设备运行状态 | HMI界面显示 |
| 报警管理 | 异常情况的报警处理 | PLC逻辑 + HMI报警显示 |
| 与打胶机交互 | 与打胶机进行通信和协调 | 硬线信号 + CC-Link总线通信 |

### 3.2 控制需求
| 控制项 | 需求描述 | 技术实现 |
|--------|----------|----------|
| 多轴控制 | X1、X2、Z三轴协调控制 | 伺服驱动器 + PLC脉冲输出 |
| 位置精度 | 定位精度±0.1mm | 高精度伺服系统 |
| 通信需求 | CC-Link总线通信 | 三菱CC-Link模块 |
| 安全保护 | 急停、限位保护 | 安全继电器 + 限位开关 |
| 与打胶机交互 | 接收打胶机允许送料信号、发送送料完成信号、允许打胶机抓料信号、安全区信号 | 硬线信号 (X76, X102, Y44, Y47) |

### 3.3 与打胶机交互详细需求
| 交互项 | 信号名称 | 信号类型 | 功能描述 | 技术实现 |
|--------|----------|----------|----------|----------|
| 打胶机输入 | 打胶机允许送料 | 数字输入 (X76) | 打胶机发出的允许送料信号 | 硬线连接 |
| 打胶机输入 | 打胶机取料完成 | 数字输入 (X102) | 打胶机发出的取料完成信号 | 硬线连接 |
| 打胶机输出 | 允许打胶机抓料 | 数字输出 (Y44) | 边框缓存机发出的允许抓料信号 | 硬线连接 |
| 打胶机输出 | 打胶机安全区信号 | 数字输出 (Y47) | 边框缓存机发出的安全区信号 | 硬线连接 |
| 通信协议 | CC-Link总线 | 总线通信 | 与打胶机进行数据交换 | CC-Link模块 |

### 3.4 交互时序需求
1. **送料流程**：
   - 边框缓存机等待打胶机允许送料信号 (X76)
   - 收到允许信号后，执行送料动作
   - 送料完成后，发出允许抓料信号 (Y44)
   - 等待打胶机取料完成信号 (X102)
   - 收到取料完成信号后，复位允许抓料信号
2. **安全区控制**：
   - 边框缓存机在安全区准备就绪时，发出安全区信号 (Y47)
   - 打胶机在安全区信号有效时，进行抓料操作
   - 边框缓存机在非安全状态时，撤销安全区信号

## 4. 非功能需求

### 4.1 性能需求
- **响应时间**：操作响应时间 < 100ms
- **运行速度**：送料速度 0-500mm/s 可调
- **可靠性**：MTBF > 5000小时

### 4.2 环境需求
- **工作温度**：0-45℃
- **工作湿度**：≤90%（无凝露）
- **供电电压**：AC220V ±10%

### 4.3 安全需求
- 符合GB/T 16855.1-2018机械安全标准
- 具备急停、过载保护功能
- 安全防护等级IP54

## 5. 数据需求

### 5.1 输入数据
| 数据类型 | 数据来源 | 处理方式 |
|----------|----------|----------|
| 传感器信号 | 光电传感器、接近开关 | PLC输入处理 |
| 操作指令 | HMI界面 | PLC程序处理 |
| 外部信号 | 上游设备 | CC-Link通信 |
| 打胶机信号 | 打胶机 | 硬线信号处理 |

### 5.2 输出数据
| 数据类型 | 输出对象 | 实现方式 |
|----------|----------|----------|
| 控制信号 | 伺服驱动器 | PLC脉冲输出 |
| 状态信息 | HMI界面 | 实时数据刷新 |
| 报警信息 | HMI界面、外部设备 | 声光报警 |
| 打胶机信号 | 打胶机 | 硬线信号输出 |

## 7. 范围限定
- 包含设备本体的控制和操作
- 包含与打胶机（下游设备）的集成
- 不包含上游设备的集成
- 不包含网络远程监控功能

## 8. 验收标准
- 设备能够正常运行30天无故障
- 定位精度符合设计要求（±0.1mm）
- 与打胶机交互功能正常，包括：
  - 正确接收打胶机允许送料信号
  - 正确执行送料动作
  - 正确发出允许抓料信号
  - 正确接收打胶机取料完成信号
  - 正确发出安全区信号
- 操作界面符合用户使用习惯
- 文档资料完整齐全
