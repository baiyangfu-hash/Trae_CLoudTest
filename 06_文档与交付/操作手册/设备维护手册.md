## 3. 维护安全注意事项

### 3.1 维护前必须执行的安全步骤
```
1. 按[停止]按钮 → 等待所有动作停止
2. 按[急停]按钮 → 切断动力电源(KM1/KM2释放)
3. 断开QF0主断路器 → 完全切断供电
4. 释放气源压力 → 打开排气阀排空管路气压
5. 挂牌上锁(LOTO) → 如有此制度要求
6. 确认无残余电压 → 用万用表测量关键点
7. 方可进入维护作业
```

### 3.2 维护后恢复步骤
```
1. 清点工具和零件 → 确认无遗留
2. 检查接线正确性 → 对照图纸核对
3. 关闭安全门 → 确认锁定到位
4. 合上QF0断路器 → 上电
5. 释放急停 → 复位报警
6. 点动测试各机构 → 确认动作正常
7. 自动运行测试 → 完整流程验证
8. 记录维护内容 → 填写维护日志
```

## 9. 校准规程

### 9.1 伺服原点校准
```
前提条件: 安全门关闭, 急停释放, 伺服无报警

步骤:
1. 进入[手动]模式
2. 选择[Z轴] → [原点回归]
   · Z轴低速向原点方向移动
   · 触发X0(原点开关) → 减速停止
   · 编码器Z脉冲确认 → 设置原点完成
   · 如果失败: 检查X0信号是否正常触发
3. 重复步骤2对X1轴(X1)、X2轴(X2)进行原点回归
4. 在HMI上确认三轴坐标显示为(0,0,0)或预设原点值
5. 做一次完整取放料动作验证原点正确性
```

### 9.2 传感器校准 (分料传感器)
```
工具: 标准边框试件(与实际产品相同尺寸)

步骤:
1. 将标准试件放在传感器检测位置
2. 进入HMI的[I/O监视]画面
3. 观察对应传感器信号状态
4. 调节传感器上的距离电位器:
   · 使信号在试件到位时可靠ON
   · 试件离开时可靠OFF
   · 留约20%的检测余量
5. 反复放置/移开试件5次, 确认每次信号正确
6. 锁定电位器并用胶封固(可选)
```

### 9.3 气缸缓冲调节
```
症状: 气缸到位时有明显撞击声

调节方法:
1. 找到气缸两端的缓冲调节阀(通常为内六角螺丝)
2. 将气缸动作一次, 注意撞击发生的末端
3. 顺时针微调该端的缓冲阀(每次1/8圈)
4. 再次动作, 直到撞击声消失但动作仍干脆
5. 注意: 不要过度缓冲导致动作不到位
```
