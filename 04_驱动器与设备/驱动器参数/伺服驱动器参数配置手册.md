### 电子齿轮计算

```
每转移动量 = 丝杠螺距 × 减速比
           = 5mm × 1 : 1 
           = 5mm/rev

每转脉冲数 = 编码器线数 × 4倍频
           = 2500 × 4
           = 10000 pulse/rev

控制精度 = 每转移动量 / 每转脉冲数 × 电子齿轮比
         = 5mm / 10000 pulse × 2000
         = 1 mm/pulse  ✓
```

## 🛡️ 制动电阻设置

### 制动电阻选型依据

```
制动功率计算：
P_brake = (1/2) × J_total × (ω²_max - ω²_min) / t_decay

其中：
- J_total：总转动惯量（电机 + 负载）
- ω_max：最高角速度
- ω_min：最终角速度（通常为0）
- t_decay：减速时间
```

### 制动电阻参数配置

| 参数名称 | X1轴 | X2轴 | Z轴 | 选型说明 |
|:---------|:----:|:----:|:---:|:---------|
| **制动电阻模式** | Pn500=3 | Pn500=3 | Pn500=1 | 3=外接, 1=内部 |
| **外接电阻功率** | 40W | 40W | 2000W | Z轴垂直负载大 |
| **外接电阻阻值** | 50Ω | 50Ω | 50Ω | 标称阻值 |
| **制动单元型号** | DBR-40W | DBR-40W | DBR-2000W | - |
| **安装位置** | 电柜内 | 电柜内 | 设备本体 | Z轴就近安装 |

### ⚠️ 注意事项

> **Z轴特殊处理**：
> - Z轴为垂直运动轴，下降时存在重力势能释放
> - 必须使用大功率外部制动电阻（≥2000W）
> - 制动电阻需安装在通风良好位置，避免过热
> - 定期检查制动电阻工作状态，防止烧毁


### 刹车控制安全联锁

```structuredtext
(* Z轴刹车控制FB *)

VAR_INPUT
    bServoReady : BOOL;      (* 伺服准备就绪 *)

    bEmgStop : BOOL;          (* 急停信号 *)

VAR_OUTPUT
    q_bBrakeRelease : BOOL;   (* 刹车释放输出 *)
END_VAR
VAR
    tBrakeDelay : TON;        (* 刹车释放延时 *)
END_VAR

(* 急停时立即抱紧刹车 *)
IF bEmgStop THEN
    q_bBrakeRelease := FALSE;
    
(* 伺服就绪后延时释放刹车 *)
ELSIF bServoReady THEN
    tBrakeDelay(IN:=TRUE, PT:=T#100MS);
    q_bBrakeRelease := tBrakeDelay.Q;
    
(* 运动完成立即抱紧 *)
ELSIF bMoveComplete THEN
    q_bBrakeRelease := FALSE;
    tBrakeDelay(IN:=FALSE);
    
ELSE
    q_bBrakeRelease := FALSE;
    tBrakeDelay(IN:=FALSE);
END_IF
END_FUNCTION_BLOCK
```
