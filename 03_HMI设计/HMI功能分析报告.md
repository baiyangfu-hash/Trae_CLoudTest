## 8. 与打胶机交互功能分析

### 8.1 交互信号

| 信号名称 | 类型 | 地址 | 说明 |
|---------|------|------|------|
| 打胶机允许送料 | 输入 | S000932 | 打胶机允许边框缓存机送料 |
| 打胶机抓料检测1 | 输入 | S000924 | 打胶机抓取边框检测1 |
| 打胶机抓料检测2 | 输入 | S000925 | 打胶机抓取边框检测2 |
| 打胶机抓料检测3 | 输入 | S000926 | 打胶机抓取边框检测3 |
| 打胶机抓料检测4 | 输入 | S000927 | 打胶机抓取边框检测4 |
| 允许打胶机抓料 | 输出 | Y44 | 边框缓存机允许打胶机抓取边框 |
| 打胶机安全区信号 | 输出 | Y47 | 边框缓存机向打胶机发送安全区信号 |

### 8.2 交互流程

1. **准备阶段**：边框缓存机准备就绪，发送安全区信号
2. **请求阶段**：打胶机发送允许送料信号
3. **送料阶段**：边框缓存机执行送料动作
4. **抓取阶段**：边框缓存机发送允许抓料信号，打胶机抓取边框
5. **完成阶段**：打胶机发送取料完成信号，边框缓存机复位
