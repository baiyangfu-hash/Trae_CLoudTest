## 4. 页面详细设计

### 4.1 登录画面
- **布局**：居中登录框，包含账号、密码输入框，登录按钮
- **功能**：用户身份验证，权限管理
- **变量**：
  - 账号输入：内部变量
  - 密码输入：内部变量
  - 登录状态：内部变量
- **操作流程**：
  1. 输入账号和密码
  2. 点击登录按钮
  3. 验证成功后进入主画面
  4. 验证失败显示错误信息

### 4.2 主画面
- **布局**：
  - 顶部：设备名称、当前时间、系统状态
  - 左侧：导航菜单
  - 中间：系统概览、设备状态
  - 底部：报警信息、快捷按钮
- **功能**：系统总览，快速导航
- **变量**：
  - 系统状态：PLC变量 (D100)
  - 报警状态：PLC变量 (D110)
  - 当前时间：系统变量
- **操作**：点击导航菜单进入相应画面

### 4.3 手动控制画面
- **布局**：
  - 顶部：轴选择标签
  - 左侧：轴控制按钮（JOG+、JOG-）
  - 右侧：气缸控制按钮
  - 底部：速度调节、传感器状态
- **功能**：手动操作各轴和气缸
- **变量**：
  - 轴JOG信号：PLC变量 (Y)
  - 气缸控制信号：PLC变量 (Y)
  - 传感器状态：PLC变量 (X)
  - 速度设定：内部变量
- **操作流程**：
  1. 选择要控制的轴
  2. 调整速度
  3. 点击方向按钮控制轴移动
  4. 点击气缸按钮控制气缸动作

### 4.4 自动运行画面
- **布局**：
  - 顶部：运行状态显示
  - 中间：启动/停止按钮
  - 底部：运行统计、当前步骤
- **功能**：控制自动运行流程
- **变量**：
  - 启动信号：PLC变量 (M100)
  - 停止信号：PLC变量 (M101)
  - 运行状态：PLC变量 (D100)
  - 运行计数：PLC变量 (D400)
- **操作流程**：
  1. 检查设备状态
  2. 点击启动按钮
  3. 观察运行状态
  4. 点击停止按钮停止运行

### 4.5 参数设置画面
- **布局**：
  - 顶部：参数类别标签
  - 中间：参数输入框
  - 底部：保存/应用按钮
- **功能**：设置系统运行参数
- **变量**：
  - 伺服参数：PLC变量 (D300-D399)
  - 工艺参数：PLC变量 (D600-D699)
  - 时间参数：PLC变量 (D700-D799)
- **操作流程**：
  1. 输入管理员密码
  2. 修改参数值
  3. 点击保存按钮
  4. 点击应用按钮使参数生效

### 4.6 状态监控画面
- **布局**：
  - 顶部：轴位置显示
  - 中间：设备状态指示灯
  - 底部：运行统计数据
- **功能**：实时监控设备状态
- **变量**：
  - 轴位置：PLC变量 (D100-D199)
  - 设备状态：PLC变量 (D100)
  - 传感器状态：PLC变量 (X)
  - 运行统计：PLC变量 (D400-D499)
- **显示内容**：
  - X1、X2、Z轴当前位置
  - 各气缸状态
  - 传感器信号状态
  - 运行次数、运行时间

### 4.7 报警管理画面
- **布局**：
  - 顶部：当前报警显示
  - 中间：历史报警列表
  - 底部：报警复位按钮
- **功能**：显示和处理报警信息
- **变量**：
  - 当前报警代码：PLC变量 (D110)
  - 报警状态：PLC变量 (M110)
  - 报警历史：PLC变量 (D500-D599)
- **操作流程**：
  1. 查看当前报警信息
  2. 检查历史报警记录
  3. 排除故障
  4. 点击复位按钮清除报警

### 4.8 打胶机交互画面
- **布局**：
  - 顶部：打胶机状态显示
  - 左侧：交互控制按钮
  - 右侧：参数设置
  - 底部：通信状态、操作日志
- **功能**：与打胶机进行通信和协调控制
- **变量**：
  - 打胶机允许送料：PLC变量 (X76)
  - 打胶机取料完成：PLC变量 (X102)
  - 允许打胶机抓料：PLC变量 (Y44)
  - 打胶机安全区信号：PLC变量 (Y47)
  - 送料请求：PLC变量 (M)
  - 通信状态：PLC变量 (M)
- **操作流程**：
  1. 查看打胶机状态
  2. 点击送料请求按钮
  3. 等待打胶机允许送料信号
  4. 观察送料过程
  5. 确认打胶机取料完成

### 4.9 机器人交互画面
- **布局**：
  - 顶部：机器人状态显示
  - 左侧：交互控制按钮
  - 右侧：参数设置
  - 底部：通信状态、操作日志
- **功能**：与机器人进行通信和协调控制
- **变量**：
  - 机器人自动中：PLC变量
  - 机器人码料完成：PLC变量
  - 机器人允许码料：PLC变量
  - 机器人停止码料：PLC变量
  - 机器人故障：PLC变量
  - 机器人急停：PLC变量
  - 通信状态：PLC变量
- **操作流程**：
  1. 查看机器人状态
  2. 点击允许码料按钮
  3. 等待机器人码料完成
  4. 观察码料过程
  5. 确认码料完成

### 4.10 系统设置画面
- **布局**：
  - 顶部：设置类别标签
  - 中间：设置项
  - 底部：保存按钮
- **功能**：系统级配置
- **变量**：
  - 系统时间：系统变量
  - 语言设置：内部变量
  - 密码设置：内部变量
- **操作流程**：
  1. 输入管理员密码
  2. 修改系统设置
  3. 点击保存按钮

## 6. 宏指令设计

### 6.1 登录宏
```
// 登录验证宏
IF UserName = "admin" AND Password = "123456" THEN
  UserLevel := 2; // 管理员权限
  CurrentPage := 2; // 跳转到主画面
ELSIF UserName = "operator" AND Password = "1234" THEN
  UserLevel := 1; // 操作员权限
  CurrentPage := 2; // 跳转到主画面
ELSE
  // 显示登录失败提示
  MessageDisplay("登录失败，请检查账号和密码");
END_IF;
```

### 6.2 报警处理宏
```
// 报警处理宏
IF AlarmCode <> 0 THEN
  // 显示报警信息
  MessageDisplay("报警：" + AlarmMessage[AlarmCode]);
  // 记录报警历史
  AlarmHistory[AlarmHistoryIndex] := AlarmCode;
  AlarmHistoryIndex := (AlarmHistoryIndex + 1) MOD 10;
END_IF;
```

### 6.3 参数保存宏
```
// 参数保存宏
// 将HMI设置的参数写入PLC
X1_Speed_PLC := X1_Speed;
X2_Speed_PLC := X2_Speed;
Z_Speed_PLC := Z_Speed;
// 显示保存成功提示
MessageDisplay("参数保存成功");
```

## 7. 画面交互设计

### 7.1 导航流程
```
登录画面 → 主画面 → 各功能画面
                    ↓
                    报警管理画面
```

### 7.2 操作流程
1. **设备启动流程**：
   - 登录系统
   - 进入手动控制画面
   - 各轴回原点
   - 切换到自动运行画面
   - 启动自动运行

2. **设备停止流程**：
   - 点击停止按钮
   - 等待设备停止
   - 退出系统

3. **报警处理流程**：
   - 系统自动弹出报警信息
   - 进入报警管理画面
   - 查看报警详情
   - 排除故障
   - 点击复位按钮

## 8. 安全设计

### 8.1 权限管理
| 权限等级 | 可访问页面 | 可执行操作 |
|----------|------------|------------|
| 操作员 | 主画面、手动控制、自动运行、状态监控、报警管理 | 基本操作，参数查看 |
| 管理员 | 所有页面 | 全部操作，参数修改 |

### 8.2 防误操作设计
- **关键操作确认**：启动/停止、参数修改等关键操作需要确认
- **权限验证**：修改参数需要管理员权限
- **操作超时**：长时间无操作自动返回主画面
- **急停按钮**：硬件急停按钮与HMI急停按钮双重保护

### 8.3 安全联锁
- 安全门打开时禁止自动运行
- 急停触发时立即停止所有动作
- 报警未复位时禁止启动

## 10. 故障排除

### 10.1 常见故障
| 故障现象 | 可能原因 | 排除方法 |
|----------|----------|----------|
| HMI无法启动 | 电源故障 | 检查电源连接 |
| 通信中断 | 通信线松动 | 检查通信连接 |
| 画面显示异常 | 程序故障 | 重启HMI |
| 触摸无响应 | 触摸屏故障 | 校准触摸屏 |
| 参数无法修改 | 权限不足 | 使用管理员账号登录 |
